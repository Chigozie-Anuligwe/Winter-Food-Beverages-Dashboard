# Winter Food & Beverages Power BI Dashboard

## Project Overview
An end-to-end Power BI dashboard analyzing winter food and beverage products across categories, countries, and seasons.

## Dataset
- Winter_Food_and_Beverages.csv

## Features
- Data cleaning using Power Query
- Data modeling for accurate analysis
- DAX measures for pricing, ratings, and popularity
- Interactive visuals for seasonal and regional insights

## Key Insights
- Drink categories generated the highest total price
- Certain products consistently ranked as top winter performers
- Popularity and calorie distribution varied across countries

## How to Use
Open the `.pbix` file in Power BI Desktop to explore the dashboard.
